(static) configRules(rules) → {Logger}

Source:
  • logger.js, line 135
Since:
  • 1.2.0

更改命名空间规则配置项
[注]从localStorage的LOGGER_RULES键中读取规则配置优先级最高，始终会覆盖其他规则

Example
// 定义规则
Logger.configRules = {
   utils-http:false // 整个utils-http不可打印输出
   utils-calc.log=true // utils-calc打印器的log方法不支持打印输出
}
