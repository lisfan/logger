rules object

打印器命名空间规则配置集合

打印器命名空间规则配置项

  • 可以配置整个命名空间是否输出日志
  • 也可以配置命名空间下某个实例方法是否输出日志
Example
Logger.rules = {
   utils-http:false // 整个utils-http不可打印输出
   utils-calc.log=true // utils-calc打印器的log方法不支持打印输出
}
